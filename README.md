# nupur-scanner-poc

Static QR scanner for the Nupur feasibility POC. Host on GitHub Pages (HTTPS) so phone cameras work on Android and iPhone.

## Live URL (after Pages is enabled)

`https://YOUR_GITHUB_USERNAME.github.io/nupur-scanner-poc/`

Replace `YOUR_GITHUB_USERNAME` with your GitHub account name.

## Files

- `index.html` — scanner page (must stay at repo root for Pages)

## Setup

See `GITHUB-PAGES-STEPS.md` in the parent folder, or follow the browser upload steps below.
